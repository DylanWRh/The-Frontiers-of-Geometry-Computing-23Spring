# Surface Reconstruction

[Tutorial](./Tutorial.md) and [report](report.md) for the assignment.

## Usage

Arrange the files as below. Run `python train.py` to train a model and `python inference.py` to do inference.
```
├── gargoyle.xyz
├── checkpoint.pth
├── inference.py
├── model.py
├── train.py
└── utils.py
```
